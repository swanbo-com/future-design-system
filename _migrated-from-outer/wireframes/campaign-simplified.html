The story · three beats

A village on the east reef, six hours from the nearest skate shop.

Siargao's skate kids ride hand-me-down decks pulled from typhoon debris. The nearest replacement is a flight away. The waves are world-class. The concrete is not.

The school rewards English class with skateboards.

Janiece runs a volunteer-led literacy class out of a bamboo pavilion. The kids who finish a module earn a deck. The decks ran out in January. The class kept going anyway.

One cascade. One budget. One story.

Every pledge buys a real thing on a real ladder. Gear → ramp → classroom → park. You'll see every receipt. You'll meet every kid. This is not a charity page — it's a commerce site with a conscience.

// wireframe v1.2

Single 4-milestone cascade. Education = milestone #3. One ledger, one narrative arc, one allocation engine.

✂ Cut from v1.1

Parallel "Skate / Education" fund split at checkout.

Removed: fund_splits JSON on Campaign, FundTrack enum, dual ledger writes, per-order split_override.

Saves ~3 dev-days.

✂ Cut from v1.1

MoU-with-school as a launch blocker.

Now a Phase 5 task with a Janiece-led LOI fallback if the formal MoU slips past Apr 30. Classroom milestone still ships; disbursement just waits on paperwork.

Saves ~1 dev-day & unblocks legal review.

// hero

Display serif (Fraunces) carries the narrative. The strike-through of "but has none left to give" is the emotional hook — it says the problem is solvable.

// progress card

Offset box-shadow + hatched accent fill = blueprint aesthetic, not SaaS. 4 milestones on a single bar reads as a ladder, not a menu.

Education milestone marked with ✎ glyph in the UI so it visibly differs from the physical builds but sits in the same cascade.

// exchange tiers

Commerce-first. Auction tier is the hero (heavier border + offset shadow), print is the volume driver (Cebu-finish + ShipBob), card is the £15 entry, name-on-the-wall is the £5 long tail.

// upsell modal

Inverted dark block = the checkout's post-purchase moment. £10/mo highlighted per conversion best-practice. Feature-flagged, can ship Day-2.

// transparency ledger

One table. Cream row = queued classroom disbursement. Backers can see the classroom is budgeted even before the cascade reaches it — that's the trust mechanism that replaces the parallel track.
