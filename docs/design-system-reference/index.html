System pattern

Title treatments — fleuron flankers

Blazeface ships 10 classical pen-flourish fleurons as 5 mirrored left/right pairs, drawn in the same hand as the Roman. They arrived orphaned in the font (no cmap entries); we patched the 18 OTFs to add entries at U+E000–U+E009. Full notes in GLYPHS.md. These are the most decorative shapes in the family — reserved strictly for the flanker slot around titles, never as inline bullets or dividers. One pair per composition. Two marks max. Restraint is what keeps them powerful.

Restraint still applies: one flanker pair per composition. If these don't feel right at small scale, they stay as inventory only — not every pair has to be used.

Primary mark — trial

Balloon floret — ❉ U+2749

Current working default. Blazeface's balloon-spoked asterisk — softer and more botanical than the 8-point star. Trials above compare it against the three other Blazeface options. Each rendering size uses the matching optical cut per GLYPHS.md. Never coloured (except vermillion in the sticker seal), never gradient, never the hero. It's punctuation.

Decision — wordmark

Logo lab — future.ly variants

The italic default read too decorative / feminine. This lab opens the field: ten treatments of future.ly, same size (48 px), same ink on cream. Pick a direction and it becomes the canonical wordmark everywhere — sticker, mailer, Lockup B, Lockup C secondary, footer. Variants are grouped by typeface family: five in Blazeface Roman (the expected home), one italic for reference, two Instrument Sans Bold, one JetBrains Mono, one alternate composition.
